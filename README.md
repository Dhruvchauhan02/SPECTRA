# SPECTRA-AI

## Overview
VeriPersona is an AI-powered platform designed to detect fake news, AI-generated images,
deepfakes, and identity misuse of public figures. The system verifies identities,
monitors public content, and alerts affected individuals in real time.

## Core Features
- Identity verification using face recognition
- AI-generated and morphed image detection
- Fake news detection using NLP
- Social media monitoring (public content only)
- Real-time alerts and verification dashboard
